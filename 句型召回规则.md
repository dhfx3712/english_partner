# 300句高考口语素材单词召回规则手册
## 一、召回目标
结合当前查询单词的**词性、使用场景、功能**，从结构化句型库中精准召回适配的句型，用于结合单词造句，实现「单词+句型」联合记忆。

## 二、匹配权重与优先级（从高到低）
完全对齐句型库标签体系，优先级越高匹配结果越精准：

| 优先级 | 匹配维度               | 匹配说明                                                                 |
|--------|------------------------|--------------------------------------------------------------------------|
| 1️⃣     | **核心标志词匹配**     | 单词词干/原形匹配句型的核心标志词标签（C类标签），直接命中对应句型        |
| 2️⃣     | **功能场景匹配**       | 单词常用交际功能匹配功能场景标签（F类标签），批量召回同功能句型           |
| 3️⃣     | **语法结构匹配**       | 单词词性/常用语法搭配匹配语法结构标签（G类标签），批量召回同语法句型      |
| 4️⃣     | **难度等级匹配**       | 单词难度匹配句型难度等级（L类标签），匹配符合当前学习阶段的句型          |
| 5️⃣     | **全文模糊语义匹配**   | 以上无匹配时，对核心句型、功能定位、例句做关键词模糊匹配，兜底召回        |

---

## 三、具体召回流程
### 步骤1：提取单词匹配信息
从单词查询结果中提取以下信息：
1. 单词原形/核心词干
2. 单词最高频的交际功能（比如：表达观点/提出建议/请求帮助/描述观点等）
3. 单词常用的语法搭配（比如：接宾语从句/用于条件句/强调句等）
4. 单词对应难度等级（对应六级体系：初中=L1→L2，高中=L2→L3，四级=L3→L4，六级/考研=L4→L6）

### 步骤2：按优先级逐级匹配
1. **第一步：精确匹配核心标志词**
   - 把单词原形/核心词干和句型库「核心标志词索引（C类）」做完全匹配，匹配成功直接召回对应句型
   - 允许多个句型命中，保留全部命中结果

2. **第二步：功能场景批量召回**
   - 若第一步匹配不足2条，根据单词的交际功能，匹配对应F类标签，补充召回同功能句型
   - 举例：单词用于「表达个人观点」→ 匹配F001标签，召回所有观点表达类句型

3. **第三步：语法结构补充匹配**
   - 若总匹配结果仍不足2条，根据单词常用语法搭配，匹配对应G类标签，补充召回同语法句型
   - 举例：单词常接宾语从句 → 匹配G005标签，召回所有含宾语从句结构的句型

4. **第四步：难度筛选**
   - 所有已召回的句型，保留和单词难度等级±1级范围内的句型，过滤难度相差过大的结果
   - 举例：单词是高中难度（L3），保留L2-L4的句型，剔除L1/L5/L6

5. **第五步：兜底模糊匹配**
   - 若前四步匹配结果仍不足2条，在全文库中模糊匹配含单词原形/同义词的句型，召回最相关的2条

### 步骤3：结果排序与输出
- 最终保留 **2-4条** 适配句型，按「精确匹配>功能匹配>语法匹配>模糊匹配」排序
- 输出格式统一为：
```
📌 适配句型（推荐{x}条）：
1. [句型ID] 核心句型：...
   功能：...
   参考结构：[替换提示：本句可填入当前单词在____位置]
```

## 五、完整召回示例
### 示例：查询单词 `convince`
1. **提取信息**：
   - 核心词干：convince
   - 高频功能：表达坚定观点 → F001（观点表达）
   - 常用语法：接宾语从句 → G005（宾语从句）
   - 难度：高考/四级 → L003，匹配范围 L002-L004

2. **逐级匹配**：
   - 第一步：核心标志词匹配 → C021 对应S117，直接命中
   - 第二步：功能匹配F001，补充召回同功能其他句型
   - 第三步：语法匹配G005，进一步补充，总结果已有4条，停止
   - 第四步：难度筛选：全部在 L002-L004，全部保留

3. **输出结果**：
```
📌 为单词 convince 召回适配句型（推荐4条）：
1. [S117] 核心句型：I’m convinced that… 我确信……
   功能：表达坚定信念与观点
   参考结构：I’m convinced that ____ (convince) everyone of the importance of environmental protection.

2. [S004] 核心句型：As far as I’m concerned, … 就我而言，……
   功能：表达个人立场与观点
   参考结构：As far as I’m concerned, we should convince students to develop good learning habits.

3. [S007] 核心句型：As I see it, … 在我看来，……
   功能：表达个人观点
   参考结构：As I see it, we can convince people to change their habits through education.

4. [S113] 核心句型：I’m certain that… 我确定……
   功能：表达确定的态度
   参考结构：I’m certain that we can convince him to join our team.
```

## 六、异常处理规则
1. 若匹配超过4条：保留优先级最高的前4条输出
2. 若完全无匹配：返回「暂无适配句型，请尝试用这个单词自己创造句子吧」，不强制兜底
3. 一词多义：优先匹配最高频的功能场景，最多召回两个不同功能场景各2条
