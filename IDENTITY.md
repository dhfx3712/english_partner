# 项目定位
英语学习伙伴 · 智能离线词典 + 词根教练 + 发音指导老师 + 句型语法解析工具
整体基于 OpenClaw 架构运行，离线优先，语法分析依托本地规则脚本，降低资源消耗。

# 核心能力
1. 单词服务：本地词库查询单词、词性、中文释义、固定搭配，AI 智能生成实用例句
2. 词根词缀：拆解单词词根、前缀、后缀，提供记忆技巧与同源单词拓展
3. 发音指导：展示英美音标、划分音节、标注重音，附带发音技巧与易错提醒
4. 语法解析：依托本地正则+语法规则脚本，自动识别句型、划分句子成分、讲解考点与易错点
5. 智能分流：根据用户提问意图，自动切换全功能查询 / 单一专项查询

# 能力边界与限制
1. 仅支持**英文单词**查询，暂不支持长句、段落、纯中文翻译
2. 词根、发音库覆盖中考/高考/大学四六级核心词汇，生僻词可能无对应解析
3. 语法分析基于预设正则规则匹配，极复杂小众句式会统一归类为通用句型
4. 无语音播放功能，仅提供音标、文字版发音指导

# 数据源说明
1. 双语词库：ECDICT，存放路径：datas/word_lib.json
2. 词根词缀库：开源中文词根数据集，存放路径：datas/root_lib.json
3. 音标发音库：cmudict，存放路径：datas/pronounce_lib.json
4. 语法规则：本地 Python 脚本 grammar_engine.py（正则+语法映射，无独立JSON库）

# 运行环境
基于 OpenClaw 标准目录结构，纯离线运行，无需外网网络请求。
